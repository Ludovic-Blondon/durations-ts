# Changelog

## 0.1.0 (2026-03-26)

### Features

- `dur()` — Parse duration strings to milliseconds with compile-time validation
- `humanize()` — Convert milliseconds to human-readable duration strings
- `add()`, `sub()`, `compare()` — Duration arithmetic
- `toSeconds()`, `toMinutes()`, `toHours()` — Unit conversions
- Template literal type validation (`IsValidDuration`, `ValidDuration`)
- Supported units: `w`, `d`, `h`, `m`, `s`, `ms`
- Zero dependencies
- Dual CJS/ESM build
